# 🛡️ Reporting a Vulnerability

The security of CareerHub Smart Job Portal is important.

If you discover a security vulnerability, please do **not** disclose it publicly.

Instead, report it responsibly by contacting the project maintainer.

Please include the following information:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)
- Screenshots or logs (if applicable)

# 🔐 Security Features

CareerHub Smart Job Portal implements several security best practices.

- JWT Authentication
- Spring Security
- BCrypt Password Encryption
- Role-Based Access Control (RBAC)
- Protected REST APIs
- Input Validation
- Global Exception Handling
- Secure File Upload Validation
- Authentication & Authorization Filters

# 🚨 Responsible Disclosure

Please allow reasonable time for security issues to be investigated and resolved before making any public disclosure.
